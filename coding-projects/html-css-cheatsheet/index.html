There are three major types of Typefaces; Serif, Sans Serif, and Monospace. The main difference between Serif and Sans Serif are these two typefaces have the presence or absence of serifs within the letters. Monospace is a font whose letters and characters each occupy the same amount of horizontal space.

font-family - Changes the type of font that is used.
font-size - Changes the size of the font.
font-weight - Changes to bold depending on value.
font-style - Changes to italic or oblique.
line-height - Changes distance between each sentence.
font-stretch - Changes how tight the letters are.
